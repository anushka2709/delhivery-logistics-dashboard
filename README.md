# 📦 Logistics Performance Dashboard

## 🔍 Overview
This project analyzes logistics delivery performance using SQL and Power BI.

## 📊 Key Features
- KPI Metrics (Total Trips, Avg Delivery Time, Delay, Efficiency)
- Center-wise performance analysis
- Trend analysis by month/year
- Delivery efficiency scatter plot

## 🛠 Tools Used
- SQL Server
- Power BI
- Python

## 📁 Files
- logistics_dashboard.pbix
- SQL query files

## 🚀 Insights
- Identified high-performing centers
- Analyzed delivery delays
- Compared actual vs expected delivery time
